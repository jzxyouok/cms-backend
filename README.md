SkeekS CMS backend
===================================

Installation
------------

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

Either run

```
php composer.phar require --prefer-dist skeeks/cms-backend "*"
```

or add

```
"skeeks/cms-backend": "*"
```

```
"repositories": [
    {
        "type": "git",
        "url":  "https://github.com/skeeks-cms/cms-backend.git"
    }
]
```
